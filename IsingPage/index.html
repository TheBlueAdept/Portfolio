Introduction

The Ising Model is a mathematical model used in statistical mechanics to describe ferromagnetism in materials.

Simulation Details

This simulation implements the 2D Ising Model using the Metropolis algorithm to study spin configurations at different temperatures. The system evolves through Monte Carlo updates, where spins flip based on the energy change they induce, following the Boltzmann probability distribution.
